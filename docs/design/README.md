# 전사도우미 디자인 문서 안내

## 목적
`docs/design` 폴더는 디자인 기준서/시안 보관용 참고 자료입니다.  
이 폴더의 문서는 현재 구현(`gui_main.py`)과 1:1 동기화 문서가 아닙니다.

## 포함 자료
- `Transcription Assistant.md`
  - 디자인 시스템 레퍼런스
- `DESIGN1.md`, `DESIGN2.md`
  - 스타일/타이포그래피/컴포넌트 레퍼런스
- `reference_main_ui.png`
  - 메인 화면 시안
- `reference_tray_notification.png`
  - 트레이/알림 시안
- `reference_app_icon.png`
  - 아이콘 시안

## 현재 구현 상태와의 관계
- 현재 구현 기능/동작의 기준 문서는 루트 `README.md`, `PROJECT_CONTEXT.md`입니다.
- 디자인 문서는 시각 참고 자료로 유지하며, 실제 동작 여부 판정 기준으로 사용하지 않습니다.
